
# cvirus

  

CLI to get corona virus updates in the terminal

  

  

 

**Install command:**

  

  

*npm install cvirus-g*

  

  

**Usage:**

  

  

cvirus *countryname*

  

  

**Options:**

  

  
  


  -V, --version  output the version number
  -a, --all      Get Summary details
  -h, --help     output usage information

  

  

**Example**: `cvirus italy` or `cvirus -a`
