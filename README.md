> Track corona virus cases in your terminal

### 🏠 [Homepage](https://www.npmjs.com/package/cvirus)

## Install

```sh
npm install cvirus-g
```

## Usage

```sh
cvirus *countryname*
```

## Example

```sh
cvirus italy  or cvirus -a  or cvirus india -d
```


## Run tests

```sh
npm run test
```
